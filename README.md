# Smart Energy Analytics Platform

Bachelor thesis project for commercial building electricity consumption analytics.

## Project Overview

This bachelor thesis project focuses on developing a smart web-based energy analytics platform for commercial building electricity consumption.

The system aims to:
- Monitor electricity usage
- Analyze energy consumption patterns
- Provide AI-driven insights
- Visualize energy data through a dashboard

## Main Features

- Energy usage visualization
- Forecasting
- Anomaly detection
- Smart recommendations
- Interactive dashboard

## Main Components

| Frontend | React |
| Backend | Node.js + Express |
| Database | MongoDB / PostgreSQL |
| Analytics | Python / JavaScript |
| Charts | Recharts / Chart.js |
